# Changelog

The changelog is generated automatically on each release from
[conventional-commit](https://www.conventionalcommits.org) messages using
[git-cliff](https://git-cliff.org/), and published with the corresponding
[GitHub Release](https://github.com/sjschlapbach/mesoLTM/releases). It is empty
until the first tagged release.
